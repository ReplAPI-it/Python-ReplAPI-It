# Python-ReplAPI-It
Python package that mirrors the original Nodejs ReplAPI-It.

## Contributing

First fork the repo:

```shell
$ git clone https://github.com/ReplAPI-it/Python-ReplAPI-It.git
$ poetry install
```

Second, edit the code. Third, clean up the project:

```shell
$ isort .
$ flake8
```

Last, create a PR!
